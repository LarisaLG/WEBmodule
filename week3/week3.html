This week we do some revision about Javascript basics such making decision, Loops and using functions. Also this week we looked at the objects.

This week we continue work with the objects. This lection covered how to create several objects to represent similar things. You create instances of the object using the constructor function. The new keyword followed by a call to the function creates a new object. The this keyword is used instead of the object name to indicate that the property or method belongs to the object that this function creates.

During this week was covered topic about Functions, making decision and Loops.

Also was covered topic about Bugs, bug detection and testing software.
